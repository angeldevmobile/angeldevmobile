### Orion Language
A modern interpreted language with clean syntax, focused on simplicity and developer experience. Supports Spanish-friendly keywords, built-in standard library (`fs`, `json`, `code`), and a custom bytecode VM.

```orion
use json
use fs

data = json.parse(fs.read("users.json"))
resumen = json.extract(data, ["nombre", "edad"])
show("Resumen:", resumen)
```

- **Built with:** Python · ANTLR4 · Custom AST  
- **Status:** Interpreter phase
